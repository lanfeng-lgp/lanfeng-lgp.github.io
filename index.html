博客two

静态博客网页

1.在终端输入：npm install hexo 安装hexo插件。

2.建立初始状态的静态博客网页

Hexo使用命令:

hexo init #执行init命令初始化hexo到你指定的目录

hexo generate #自动根据当前目录下文件,生成静态网页

hexo server #运行本地服务,

浏览器输入http://localhost:4000
就可以看到效果

3.添加博文

hexo new “postName” #新建博客,其中postName是博客题目

博客会自动生成在博客目录下source/_posts\postName.md

4.网页生成

hexo g #生成静态网页

注：在Hexo 3.0版本后deploy git 被分开的,导致部署提示找不到Git ，要加上npm install hexo-deployer-git –save

hexo d #开始部署

博客one

同步线上博客

1.在https://github.com/
中生成SSHkey

在终端中找到当前目录填入：sh-keygen #生成密钥，判断的要时yes,其中密码为空。

继续填入：vim ~/.ssh/id_rsa.pub #显示秘钥，添加到SSH keys中

2.给本地配置

终端中填入：git config –global user.name “Your Name”#配置用户名

git config –global user.email “email@example.com()mailto:email@example.com)”#配置邮箱

2.修改下载文件，同时同步线上内容

git clone#下载线上仓库地址

git status#查看当前的仓库状态

git add .#生成快照并存入项目索引

git commit #提交到暂存区

git remote add origin git@github.com:yourname/learngit.git#添加远程仓库

git push -u origin master #推送本地更新到远程
