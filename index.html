伪类与伪元素


伪类用于向某些选择器添加特殊的效果。效果可以通过添加一个实际的类来达到
伪元素用于将特殊的效果添加到某些选择器或者通过添加选择器来实现效果。效果则需要通过添加一个实际的元素才能达到

伪类的使用循序：:link -> :visited -> :hover ->:active

伪元素

这两个伪元素相当于添加\标签
content:”” 伪元素before,after下特有的属性,必须添加，用于在 CSS 渲染中向元素逻辑上的头部或尾部添加内容。
注意这些添加不会改变文档内容，不会出现在 DOM 中，不可复制. “”中可以是添加的文本内容，可以是空。

具体操作：

1.注册git,设置你的用户名称和e-mail地址

设置你的用户名称和e-mail地址。因为每次Git提交都会使用该信息。它被永远的嵌入到了你的提交中;只需要做一次这个设置

2.服务器上的Git - 生成SSH公钥

在github中生成SSHkey公钥

ssh-keygen 会确认密钥的存储位置（默认是 .ssh/id_rsa），然后它会要求你输入两次密钥口令。如果你不想在使用密钥时输入口令，将其留空即可。
vim ~/.ssh/id_rsa.pub 显示.pub 文件内容（即key）

3.创建新仓库

4.添加与提交

如果你还没有克隆现有仓库，并欲将你的仓库连接到某个远程服务器，你可以使用如下命令添加：git remote add origin git@github.com:yourname/learngit.git
